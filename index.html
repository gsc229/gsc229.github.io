Market Ave. Api

The Market Ave. e-commerce app is a sales platform for vendors of farmers' markets. I used Node, Express, Mongo, and Mongoose to build the backend.

With this robust api you can browse vendors within any distance of a zip code, query products by their category, populate vendor objects with an array of all their product-objects and much more (checkout advanced filtering).

Link to my API documentation for this project:
Market Ave API documentation

Feel free to read the docs and make some queries using this base URL: https://quickstlabs.herokuapp.com/api/v1.0
If you query by zip code, you can use 18641 as this was our test zip code in development and there are plenty of vendors around. (Note: if the server has been inactive for a while, your first query may take a few seconds)

Using React, I built the vendor profile section for the front end. You can sign in and try it out using the test profile:
username: testvendor@mail.com
password: 123456

You can line-item edit the vendor's information by clicking directly on the information. Feel free to create a new product if you want. Use the upload widget to upload photos from a variety of sources: your local drive, Facebook, live camera shot, or image address from a google image search.

For image management, I implemented the Cloudinary image management platform. The Cloudinary API provides users a wide range of options to customize their profiles through multiple image upload sources, cropping, and transforming their images on the fly.

Market Ave. Api

The Market Ave. e-commerce app is a sales platform for vendors of farmers' markets. I used Node, Express, Mongo, and Mongoose to build the backend.

With this api you can browse vendors within any distance of a zipcode, query products by their category, populate vendor objects with an array of all their product-objects and much more.

Link to my API documentation for this project:
Market Ave API documentation

Feel free to read the docs and make some queries using this base URL: https://quickstlabs.herokuapp.com/api/v1.0
If you query by zipcode, you can use 18641 as this was our test zipcode in development and there are plenty of vendors around.

Using React, I built the vendor profile section for the front end. You can sign in and try it out using the test profile:
username: testvendor@mail.com
password: 123456

You can line-item edit vendor's information by clicking directly on the information. Feel free to create a new product if you want. Use the upload widget to upload photos from a variety of sources: your local drive, Facebook, live camera shot or image address from a google image search.

For image management, I implemented the Cloudinary image management platform. This option offers users a wide range of options to customize their profiles through multiple image upload sources, cropping and modifiying their images.
